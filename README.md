# AWS S3 Static Website Hosting

## Overview
This project demonstrates how to host a static website using AWS S3, including configuring public access and enabling static website hosting.

## Live Website
http://ivan-s3-static-website-lab1.s3-website-us-east-2.amazonaws.com
Note: Public access was enabled via bucket policy and block public access settings to allow the site to be accessible.

## Technologies Used
- AWS S3
- Static Website Hosting
- HTML
- Cloud Storage

## What I Did
- Created an S3 bucket
- Uploaded HTML files for the website
- Enabled static website hosting
- Configured bucket policy for public access
- Disabled block public access settings
- Verified the website is accessible via S3 endpoint

## Key Concepts Learned
- How S3 can be used for static website hosting
- How bucket policies control access
- Difference between private and public S3 objects

## Real-World Use Case
This setup can be used to host portfolio websites, landing pages, or static web applications without needing a server.
